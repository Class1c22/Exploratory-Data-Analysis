What is the project about?
The project aims to explore a dataset related to Telegram channels used for Russian propaganda.

Why is this project useful?
This project addresses questions that the author investigates based on the data in the dataset. For the user, this research is valuable as it provides insights into how propaganda works, allowing the reader to draw their own conclusions.

How to read the research?
Simply open the Jupyter Notebook file and review the content.
